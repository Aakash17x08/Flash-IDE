# Flash IDE

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Flash IDE** is a lightweight, AI-powered online code editor designed to enhance productivity for developers. It features real-time code editing, AI-assisted code generation, and multi-language support, providing a modern and interactive coding experience directly from your browser.

## Features

- **Rich Text Code Editor** with syntax highlighting for multiple programming languages.
- **AI Code Generation**: Generate code using the custom AI backend (`Gemini API`).
- **Multi-Language Support**: HTML, CSS, JavaScript.
- **Real-Time Preview**: See the output of your code instantly.
- **Lightweight & Fast**: Optimized for speed and minimal resource usage.

---

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **AI Integration**: Custom API (`/api/gemini`) 
- **Others**: Axios, CORS, dotenv
